Navegación

¡Es muy sencillo navegar por nuestra aplicación!
Solo selecciona el ícono asignado a la sección que quieres visitar.

El Dashboard: La joya de nuestra app

El Dashboard está diseñado para visualizar los datos que se utilizaron para realizar las predicciones a futuro del estilo de vida de Colombia.
Puedes interactuar con los siete tipos de gráficos que ofrecemos.

Usar un Linechart

Para observar las diversas inversiones realizadas por el mintic y el índice de calidad de vida por departamento, solo seleccionas el departamento que quieres visualizar.
Si se pone el cursor encima de cada punto, se puede ver el valor preciso de las inversiones y el índice de calidad de vida.
Si mueves los valores a la izquierda y la derecha de la gráfica, puedes desplazar las lineas.

Usar un Stripplot

Selecciona el eje horizontal y vertical para visualizar distintos stripplots.
Ubica el cursor encima de los puntos para ver información acerca de ese punto.

Usar un Geograph

Selecciona el año y el tipo de inversión para ver en el mapa cómo se distribuyen las inversiones del mintic en Colombia.

Usar un Bargraph

Selecciona el eje horizontal y el eje vertical para visualizar las distintas cantidades en un diagrama de barras.

Usar un Treemap

Selecciona el eje horizontal y el eje vertical para visualizar las distintas cantidades en un treemap.
Ubica el cursor encima de cada caja para ver información adicional.

Usar un Piechart

Selecciona el eje horizontal y el eje vertical para visualizar las distintas cantidades en un diagrama de tortas.
Ubica el cursor encima de cada sección para ver el valor exacto en vez del porcentaje.

Usar un Distplot

Selecciona el eje horizontal y el eje vertical para visualizar las distintas cantidades en un treemap.
Ubica el cursor encima de cada columna y del gráfico superior para ver información relacionada con cuartiles, medias y medianas de la distribución.

¿Qué es esta aplicación?

Intentando predecir la calidad de vida para el futuro

Es una aplicación que tiene como fin último mostrar los resultados de las predicciones de nuestros modelos de la calidad de vida para el futuro según las inversiones que ha hecho el MinTic.
En la sección de Descripción de Modelos de la app se puede encontrar una descripción técnica de los modelos utilizados.
Por último, en la sección de interpretación y recomendaciones del equipo se encuentran nuestras conclusiones e interpretaciones hechas por el equipo de desarolladores a partir del análisis de datos.
