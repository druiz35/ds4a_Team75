XGB Boost

XGBoost es una biblioteca optimizada de aumento de gradiente distribuida diseñada para ser altamente eficiente, flexible y portátil. Implementa algoritmos de aprendizaje automático bajo el marco Gradient Boosting.
XGBoost ofrece muchas herramientas para Machine Learning, una de ellas es el modelo de Regresión, implementado en este cuaderno, que nos permite generar un modelo de predicción, y da luz sobre cuáles podrían ser las principales variables y su importancia para hacer una predicción sobre la variable. elegimos como representación de la Calidad de Vida.
La forma en que funciona el modelo es generando un árbol de decisión a partir de la etapa del tren y el uso de este árbol para realizar una predicción con los datos de prueba o cualquier dato que tenga las características involucradas.

OLS Model

Para el modelado utilizamos datos de 2016 a 2018, utilizando las características que encontramos con información completa para todos los departamentos. Luego, eliminamos las características que estaban altamente correlacionadas, se desintegraron en otras características o no eran relevantes para todos los departamentos y procedimos a hacer un modelo inicial. Usamos errores robustos de HC3 para dar cuenta de una posible heterocedasticidad en la varianza de los errores.

Luego procedimos a reducir aún más las características eliminando las menos significativas una por una. Empleamos el 75% del conjunto de datos como entrenamiento y el 25% como prueba para el modelo. Por el simple hecho de considerar la importancia de las funciones de inversión, no las eliminamos aunque no fueran significativas.
